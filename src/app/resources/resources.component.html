This calculator calculates the BMI (Body Mass Index) which indicates if the person is applicable to any of the following classifications:

  • Underweight: BMI < 18.5
  • Normal: 18.5 <= BMI <= 24.9
  • Overweight: 25 <= BMI <= 29.9
  • Obese: 30 <= BMI <= 34.9
  • Extremely Obese: 35 < BMI
This classification is based on the information that has been featured on the CDC website here.

The calculator's results have been tested and compared to the results produced by Diabettes Canada website here
